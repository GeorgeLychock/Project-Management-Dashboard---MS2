Welcome to pmDashboard!
  • pmDashboard is a concept for a configurable project management dashboard.

  • There are 5 pre-populated projects available in the Library.

  • You can Add or Delete your own projects, they will be saved locally in your browser.

  • Use the Library menu to Add/Remove project panels on the dashboard.

  • There are two available Tools widgets: OpenWeather (you can configure the location), and GitHub (you can search for a valid GitHub user).

  • If you log in (this is simulated, there is no backend), you will be presented the user panel.

  • You can configure a few user settings by clicking the cogs.
